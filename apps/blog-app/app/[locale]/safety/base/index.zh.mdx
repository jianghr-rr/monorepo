## Web app

简单的流程
DNS - IP - 服务器 - 建立 TCP/IP 连接 - 客户机发送 HTTP报文 - 服务器返回响应报文

1. 请求的种类
html/php/数据库
- HTML
    客户机（浏览器）直接解析文档
- php
    如果是这种动态的，会转成html返回给客户机
- 访问数据库
    php应用服务器处理再去访问数据库

2. 常见的web服务器

- Nginx
- IIS
- Lighttpd
