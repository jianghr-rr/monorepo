# groupAnagrams

哈希

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是由重新排列源单词的所有字母得到的一个新单词。

**示例 1:**

输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出: [["bat"],["nat","tan"],["ate","eat","tea"]]

**示例 2:**

输入: strs = [""]
输出: [[""]]

**示例 3:**

输入: strs = ["a"]
输出: [["a"]]

``` javascript
/**
 * @param {string[]} strs
 * @return {string[][]}
 */
var groupAnagrams = function(strs) {
    const map = new Map();
    
    for (let i of strs) {
        // 生成模版，模版作为map的key
        let temp = [...i].sort().join('');

        // 如果之前存在相同的模版，就加入这个key的数组
        // 否则就新建一个
        if (map.has(temp)) {
            map.get(temp).push(i);
        }
        else {
            map.set(temp, [i])
        }
    }

    return [...map.values()];
};
```

`map.values()`返回的是一个迭代器对象，可以通过解构生成数组
