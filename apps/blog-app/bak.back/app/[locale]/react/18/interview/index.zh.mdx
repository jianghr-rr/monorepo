# React 参考概述

## React

- Hooks - 使用组件中的不同 React 功能。
- 组件 - 记录可在 JSX 中使用的内置组件。
- API - 用于定义组件的 API。
- 指令 - 向与 React Server 组件兼容的打包器提供指令。

## React DOM

- Hooks - 用于在浏览器 DOM 环境中运行的 Web 应用程序的钩子。
- 组件 - React 支持所有浏览器内置的 HTML 和 SVG 组件。
- API - 该 react-dom 包包含仅在 Web 应用程序中支持的方法。
- 客户端 API - 这些 react-dom/client API 允许您在客户端（在浏览器中）渲染 React 组件。
- 服务器 API - 这些 react-dom/server API 允许您在服务器上将 React 组件呈现为 HTML。

## Rules of React

- 组件和钩子必须是纯的 – 纯度使你的代码更易于理解、调试，并允许 React 自动正确地优化你的组件和钩子。
- React 调用组件和钩子 – React 负责在必要时渲染组件和钩子，以优化用户体验。
- 钩子规则 – 钩子是使用 JavaScript 函数定义的，但它们表示一种特殊类型的可重用 UI 逻辑，但对它们的调用位置有限制。
