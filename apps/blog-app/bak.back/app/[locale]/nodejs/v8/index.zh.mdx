# V8简介

Chrome 的 V8 引擎

JavaScript 引擎是一个将JavaScript 代码转换为机器代码，允许计算机简单地执行特定任务， JavaScript 引擎可以执行 JavaScript 代码

JavaScript 引擎通常由 Web 浏览器供应商开发，以下是每个主要浏览器的不同 JavaScript 引擎的列表浏览器

- V8，它是Google 为 Chrome 开发的开源 JavaScript 引擎
- SpiderMonkey， Firefox
- JavaScriptCore， Safari
- Chakra， 原始 Microsoft Edge，现在用的V8

## 简介

V8 is Google's open source JavaScript engine.

V8 implements ECMAScript as specified in ECMA-262.

V8 is written in C++ and is used in Google Chrome, the open source browser from Google.

V8 can run standalone, or can be embedded into any C++ application.
