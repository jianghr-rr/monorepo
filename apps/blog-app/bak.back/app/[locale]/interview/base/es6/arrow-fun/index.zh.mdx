## 箭头函数的参数特性  

arguments

``` javascript
const obj = {
    name: 'obj',
    getName: () => {
        console.log(this.name);
    }
}
```

**dom操作的callback**
``` javascript
const $a = document.getElementById('a');

a.addEventListener('click', () => {
    this.style.color = 'red';
})
```
